In the field of mathematics, I'm mostly interested in the foundations of mathematics, set theory and ordinal analysis. My favourite number is 𝔡 the dominating number! (are infinite numbers allowed?) The dominating number is a cardinal characteristic of the continuum: it lies between the second least infinite cardinal number (a cardinal number measures how many elements are in a set), ℵ1, and the continuum, 𝔠 (the cardinality of the set of real numbers). ℕ → ℕ denotes the set of functions from natural numbers to natural numbers. Functions f, g: ℕ → ℕ can be compared using ≤*: f ≤* g iff there exists a natural number N so that, for all n > N, f(n) ≤ g(n). The dominating number is the size of the smallest set of functions F so that, for all f: ℕ → ℕ, there is some g ∈ F so that f ≤* g.
